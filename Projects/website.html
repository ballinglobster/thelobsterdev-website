Portfolio Website

This is my personal portfolio website, the website you are on right now! I designed and developed this site to showcase my projects, skills, and background in game design and development. It is built using HTML, CSS, and JavaScript completely from scratch.

You can find the source code on GitHub.

What I Did

I was responsible for the entire design and development of this website. This includes:

  • Creating the overall layout and structure using HTML
  • Styling the website with CSS to achieve a visually appealing design
  • Implementing interactivity and animations using JavaScript
  • Ensuring the website is responsive and works well on different devices
  • Optimizing performance and accessibility for a better user experience
  • Adding smooth page transitions and animations

What I Learned

Creating this website has been a valuable learning experience for me. I improved my skills in web development, including HTML structure, CSS styling, and JavaScript functionality. I also learned about responsive design to ensure the site looks great on various devices and screen sizes. Additionally, I gained experience in optimizing website performance and accessibility.

Overall, building this portfolio website has been a rewarding project that has helped me improve my web development skills and showcase my work effectively.
