"HA: Food World" is a conceptual mobile game to help new and upcoming parents find quick and easy recipes for their families. It was developed as a team project during my studies using Unity and C#.

My contributions

Several core programming features:

  • Recipe unlocking
  • Customer interactions and feedback
  • The tutorial system
  • Level progression

Assets:

  • Several UI assets
  • Several 3D assets

What I Learned

Working on "HA: Food World" allowed me to enhance my skills in Unity and C# programming, particularly in implementing game mechanics and user interactions.
I also gained experience in 3D asset creation using Blender, improving my modeling and texturing abilities. Collaborating with a team helped me develop better communication and project management skills, as we were using the SCRUM/Agile methodology.
